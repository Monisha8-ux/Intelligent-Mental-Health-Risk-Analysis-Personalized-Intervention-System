# 🧠 Intelligent Mental Health Risk Analysis & Personalized Intervention System

## 📌 Overview

This project is a Machine Learning-based system designed to analyze mental health conditions using user behavioral and psychological data. It predicts the risk level (Low, Medium, High) and provides personalized intervention suggestions.

The goal is to support early awareness and assist users in understanding their mental health patterns using data-driven insights.

---

## 🎯 Features

* Mental health risk prediction (Low / Medium / High)
* Data preprocessing and feature engineering
* Machine Learning model training and evaluation
* Real-time prediction system
* Simple and interactive UI (Streamlit)
* Personalized intervention suggestions

---

## 🛠️ Tech Stack

* Python
* Pandas, NumPy
* Scikit-learn
* Streamlit
* Matplotlib / Seaborn

---

## ⚙️ How It Works

### 1. Data Preprocessing

* Handling missing values
* Encoding categorical variables
* Feature selection

### 2. Model Training

* Algorithm used: Random Forest / Decision Tree
* Train-test split applied
* Model trained on structured dataset

### 3. Prediction

* User inputs data via UI
* Model predicts risk level
* Output displayed with suggestions
